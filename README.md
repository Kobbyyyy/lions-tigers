Infinite Dog App

Author: Nana Kwabena Inkum
Project: Infinite Dog App - A JavaScript application that fetches and displays dog images from the Dog CEO API

Overview

This application allows users to select different dog breeds and view an infinite slideshow of adorable dog photos. It demonstrates asynchronous JavaScript, Fetch API, and dynamic DOM manipulation.

Live Demo

The application is hosted on Azure Static Web Apps:
https://brave-water-06a30fe1e.1.azurestaticapps.net

Credits

This application was developed following the tutorial by Brad Schiff. Full credit for the original concept and implementation guidance goes to Brad Schiff.


- Tutorial Reference: Dogs, JavaScript & An API course

Features

- Fetch and display list of all dog breeds from Dog CEO API
- Select a breed to load its images
- Automatic slideshow with infinite scrolling effect
- Responsive design for mobile and desktop

Technologies Used

- HTML
- CSS 
- JavaScript 
- Dog CEO API 
- Azure Static Web Apps for hosting

Prerequisites

- Google Chrome
- Internet connection for API calls
- Code editor (for local development)
